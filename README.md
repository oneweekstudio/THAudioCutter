## Installation

THAudioCutter is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'THAudioCutter', :git => 'https://github.com/oneweekstudio/THAudioCutter.git'
```
